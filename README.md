  4 (연습문제 1.3-1) 쌍곡함수
* 서론 (문제 인식)
쌍곡함수인 $\sinh x$와 $\cosh x$는 지수함수 $e^x$를 이용하여 정의된다. 문제에서는 쌍곡선함수에 로그값이 입력값으로 주어져서 정확한 값을 알아내기 힘들다. 쌍곡함수의 정의를 적용하여 복잡한 지수 형태의 식을 계산해본다.
* 본론 (수학적 조작)쌍곡함수의 정의를 이용한다. $\sinh x = \frac{e^x - e^{-x}}{2}$ $\cosh x = \frac{e^x + e^{-x}}{2}$
  $\sinh(\ln 2)$ 계산: $\sinh(\ln 2) = \frac{e^{\ln 2} - e^{-\ln 2}}{2} = \frac{2 - \frac{1}{2}}{2} = \frac{\frac{3}{2}}{2} = \frac{3}{4}$
  $\cosh(\ln 3)$ 계산: $\cosh(\ln 3) = \frac{e^{\ln 3} + e^{-\ln 3}}{2} = \frac{3 + \frac{1}{3}}{2} = \frac{\frac{10}{3}}{2} = \frac{5}{3}$


두 값을 더하면 $\frac{3}{4} + \frac{5}{3}$이고 $\frac{29}{12}$가 된다.

* 결론 (결과 해석)
쌍곡함수는 현수선(Catenary)의 방정식을 기술하거나 물리학에서의 파동 전파 등을 분석할 때 핵심적으로 사용된다. 복잡한 초월함수 모델을 산술적인 수치로 환원하여 공학적 설계에 반영할 수 있음을 보여준다

5 (연습문제 1.4-6) 역삼각함수
* 1단계 (What): 계산 과정 및 근거 역함수의 각도 정의는 특정 삼각비에 대응하는 각을 설정함을 보여준다. $\cos^{-1}(4/5) = \alpha$, $\cos^{-1}(3/5) = \beta$라고 하면 $\cos\alpha = 4/5$, $\cos\beta = 3/5$가 성립함을 알 수 있다.삼각항등식($\sin^2\theta + \cos^2\theta = 1$)은 나머지 사인 값을 구하는 결정적 근거를 제공한다. 이를 통해 $\sin\alpha = 3/5$, $\sin\beta = 4/5$라는 구체적인 수치를 얻는다.삼각함수의 덧셈정리는 두 각의 합에 대한 사인 값을 단일 수치로 산출하는 과정을 보여준다.  $\sin(\alpha+\beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$ 식에 대입하면 $(3/5 \times 3/5) + (4/5 \times 4/5) = 25/25 = 1$이라는 결과가 나온다.
* 2단계 (So What): 두 역삼각함수의 합($\alpha + \beta$)이 정확히 $90^\circ$($\pi/2$)인 직교 상태임을 보여준다. 이는 $3, 4, 5$ 비율을 가진 직각삼각형에서 두 예각의 합이 직각을 이루는 기하학적 대칭성을 증명한다.이러한 수치적 일치는 복잡한 위상차를 가진 신호들이 결합하여 에너지 효율이 극대화되는 지점을 명확히 제시함을 보여준다.

  
6 (예제 1.4-15) 역쌍곡사인함수
* 1단계 (What): 계산 과정 및 근거역함수의 정의는 $x$와 $y$의 역할을 교환하여 원래의 함수 관계로 되돌리는 과정을 보여준다. $y = \sinh^{-1}x$라고 하면 $\sinh y = x$가 성립하며, 이는 쌍곡함수의 정의에 의해 $\frac{e^y - e^{-y}}{2} = x$로 치환됨을 의미한다.지수 방정식의 풀이는 복잡한 초월함수를 이차방정식의 형태로 변환할 수 있음을 보여준다. 식을 $e^y$에 대해 정리하기 위해 양변에 $2e^y$를 곱하면 $(e^y)^2 - 2xe^y - 1 = 0$이라는 $e^y$에 관한 이차식을 얻는다.근의 공식과 로그의 정의는 최종적으로 $y$ 값을 $x$에 관한 식으로 도출하는 근거를 제공한다. 근의 공식을 적용하면 $e^y = x \pm \sqrt{x^2+1}$이 나오는데, $e^y > 0$이어야 하므로 $e^y = x + \sqrt{x^2+1}$만을 선택한다. 양변에 자연로그를 취하면 $y = \ln(x + \sqrt{x^2+1})$이 도출된다.
* 2단계 (So What): 결과 해석 및 공학적 통찰이 증명은 쌍곡함수의 역관계가 대수적으로 로그 함수와 연결되어 있음을 명확히 보여준다. 이는 기하급수적으로 증가하거나 감소하는 물리적 현상을 해석할 때, 역으로 그 원인이 되는 변수를 로그 단위로 정밀하게 추적할 수 있음을 의미한다.로그 표현식으로의 변환은 공학 계산기나 소프트웨어에서 역쌍곡함수를 효율적으로 수치 해석하는 토대가 됨을 보여준다.

 7 (연습문제 3.1-1) 연속이며 미분 가능한 함수 
* 1단계 (What):  $x \to 0$일 때 $-x^2 \le x^2\sin(1/x) \le x^2$이 성립하며, 조임정리(Sandwich Theorem)에 의해 $\lim_{x\to 0} f(x) = 0$이 도출된다. 이는 $f(0)=0$과 일치하므로 원점에서 연속임을 증명한다.미분 가능성 조사는 도함수의 정의를 이용한 극한의 수렴성을 근거로 삼는다. $f'(0) = \lim_{h\to 0} \frac{h^2\sin(1/h) - 0}{h} = \lim_{h\to 0} h\sin(1/h)$로 식을 정리한다.절댓값의 유계성을 이용한 극한 계산은 특정 점에서의 미분계수 존재를 보여준다. $|h\sin(1/h)| \le |h|$이므로 $h \to 0$일 때 미분계수는 $0$으로 수렴하며, 따라서 $x=0$에서 미분 가능하다는 결론을 얻는다.
* 2단계 (So What): 결과 해석 및 공학적 통찰이 분석 결과는 진동이 극심한 신호라도 진폭의 감쇄 속도가 빠르면 시스템의 연속성과 매끄러움이 유지됨을 보여준다. 수학적 엄밀함을 통한 미분 가능성 증명은 물리적 시스템의 예측 가능성을 확보하는 토대가 됨을 보여준다. 단순히 그래프의 형태만으로 판단하지 않고 정의에 입각해 분석함으로써, 불연속적인 점에서의 오작동을 방지하는 엔지니어의 분석적 사고를 뒷받침한다

 9 (연습문제 3.2-2) 음함수의 고계 도함수I
* 1단계 (What): 계산 과정 및 근거음함수의 미분법은 $y$를 $x$의 함수로 간주하여 양변을 미분하는 과정을 보여준다. 곡선 $2x^2y - 4y^3 = 4$의 양변을 $x$에 대해 미분하면 $4xy + 2x^2y' - 12y^2y' = 0$이 된다.점 (2, 1)의 대입은 특정 지점에서의 기울기($y'$)를 산출하는 근거를 제공한다. $x=2, y=1$을 대입하여 정리하면 $8 + 8y' - 12y' = 0$이 되므로, $y' = 2$임을 얻는다.이계 도함수의 도출은 $y'$가 포함된 식을 한 번 더 미분하여 곡률을 계산하는 단계를 보여준다 앞선 미분식을 다시 미분하고 $x=2, y=1, y'=2$를 대입하여 정리하면 $y''$의 값을 최종적으로 산출할 수 있다
* 2단계 (So What):  $y'$와 $y''$는 해당 점에서의 접선의 기울기와 곡선의 휘어짐 정도를 명확히 보여준다 . 이는 함수가 $y=f(x)$ 형태로 정의되지 않더라도 특정 위치에서의 변화율과 가속도를 정밀하게 분석할 수 있음을 의미한다.음함수 분석 기술은 설계 복잡도가 높은 공학적 궤적이나 물리적 경계면의 안정성을 평가하는 토대가 됨을 보여준다

 10 (연습문제 3.2-4) 음함수의 고계 도함수II
* 1단계 (What) 
곡선 $x^2 + 2y^3 = xy + 9$는 $y$가 $x$에 대한 함수로 직접 표현되지 않는 음함수 형태다. 따라서 양변을 $x$에 대해 미분하는 음함수 미분법을 적용하여 도함수 $y'$와 고계 도함수 $y''$를 구한다 1계 도함수 ($y'$) 구하기양변을 $x$에 대해 미분하면 다음과 같다. $$2x + 6y^2 \cdot y' = y + x \cdot y'$$ 점 $(3, 0)$을 대입하면: $$2(3) + 6(0)^2 \cdot y' = 0 + 3 \cdot y'$$ $$6 = 3y' \quad \therefore y' = 2$$  2계 도함수 ($y''$) 구하기위에서 구한 미분식 $2x + 6y^2 y' = y + xy'$를 한 번 더 $x$에 대해 미분합니다 (곱의 미분법 적용) $$2 + 12y(y')^2 + 6y^2 y'' = y' + y' + xy''$$ 점 $(3, 0)$과 앞서 구한 $y' = 2$를 대입하면: $$2 + 12(0)(2)^2 + 6(0)^2 y'' = 2 + 2 + 3y''$$ $$2 = 4 + 3y'' \quad \therefore y'' = -\frac{2}{3}$$
* 2단계 (So What): 기하학적 의미 (접선의 기울기와 곡률)
점 $(3, 0)$에서 $y'=2$라는 결과는 이 곡선이 해당 지점에서 우상향하는 직선의 기울기를 가짐을 의미한다. 또한, $y'' = -2/3$로 음수 값이 도출된 것은 해당 점 근방에서 곡선이 **위로 볼록(Concave down)**한 형태를 띠고 있음을 나타낸다.
