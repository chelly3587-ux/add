 1 (연습문제 1.1-3) 합성함수의 정의역
* 1단계 (What): $f(g(x)) = \sqrt{1/x+1}$, 정의역은 $1/x+1 \ge 0$ 이고 $x \ne 0$이므로 $x \le -1$ 또는 $x > 0$이다. $g(f(x)) = 1/\sqrt{x+1}$, 정의역은 $x+1 > 0$이므로 $x > -1$이다.
* 2단계 (So What): 합성함수의 정의역은 내부 함수의 치역이 외부 함수의 정의 영역 내에 포함되어야 함을 보여준다.

 
 2 (연습문제 1.1-8) 합성함수 형태로 표현
* 1단계 (What): $g(x) = |x^3-1|+3$, $f(x) = \sqrt{x}$로 두면 $F(x) = f(g(x))$로 표현된다
* 2단계 (So What): 복잡한 함수를 기본 함수의 결합으로 분해함으로써 미분이 용이하다.

  
 3 (연습문제 1.2-8) 자연로그 역함수
* 1단계 (What): 함수 $y = e^{x-1}$ 의 도함수는 원함수와 같으므로 모든 실수 x에 대해 양수이다. 그러므로 함수$e^{x-1}$는 증가함수이고 일대일 대응이므로 역함수의 존재조건을 만족한다. $x-1 = \ln y$이므로 $x = \ln y + 1$이다. 따라서 $f^{-1}(x) = \ln x + 1$ (단, $x > 0$)이다
* 2단계 (So What): 지수함수의 치역이 역함수의 정의역이 되고, $y=x$에 대해 대칭인 그래프 특성을 가진다.

  4 (연습문제 1.3-1) 쌍곡함수
 * 1단계 (What): $\sinh(\ln 2) = (2 - 1/2)/2 = 3/4$, $\cosh(\ln 3) = (3 + 1/3)/2 = 5/3$. 합은 $3/4 + 5/3 = 29/12$이다.
 * 2단계 (So What): 쌍곡함수는 지수함수의 조합으로 정의되며, 현수선 등의 물리적 모델링에 필수적이다.
